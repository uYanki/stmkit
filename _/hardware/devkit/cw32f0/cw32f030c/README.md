**特点：**

l MCU: CW32F030C8T6 ,ARM®Cortex®-M0+ 32 位微控制器，64K 字节 FLASH，8K 字节 RAM,LQFP48

l 4针SWD下载接口；

l 一路USART接口。

l 一路电位器

l 三路指示灯，三个独立按键、一个复位键

l 0.91OLED显示屏；

l 一个蜂鸣器

l EG3013驱动，6路MOS控制

l 单电阻电流采样

l 一路NTC温度传感器

l 一路扩展电机霍尔及UVW接口;

l 一个专用电机接口

l 一路DC电源接口

l 套件配置：一套下载器、一个主板、一台电机、一个电源
